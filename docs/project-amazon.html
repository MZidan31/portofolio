Analisis Laporan Penjualan Amazon

Dibuat oleh: Muhammad Zidan

1. Business Understanding

Masalah utama dari data ini adalah tidak terstrukturnya data penjualan sehingga menyulitkan pemilik perusahaan dalam pengambilan keputusan, merancang strategi pemasaran, dan meningkatkan pendapatan.

Analisis data ini bertujuan untuk memetakan dan memahami penjualan Amazon secara menyeluruh, serta menemukan wawasan tersembunyi yang dapat digunakan sebagai rekomendasi bagi perusahaan dalam mengoptimalkan strategi penjualan mereka di masa depan.

2. Data Understanding

Dataset yang digunakan berjudul “Sales Report.csv”. Dataset ini terdiri dari 13 kolom dengan rincian berikut:

Kolom Keterangan Tipe Data
Date Tanggal penjualan Date
Status Status pembelian String
Fulfilment Metode pengiriman String
Sales Channel Saluran penjualan String
ship-service-level Tingkat layanan pengiriman String
category Kategori produk String
size Ukuran produk String
Qty Jumlah produk Integer
Amount Jumlah penjualan Numeric

3. Data Preparation

Tahapan persiapan data dilakukan menggunakan KNIME Analytics Platform dengan langkah-langkah berikut:

  1. Load data menggunakan CSV Reader.
  2. Hapus kolom `Unnamed: 22` menggunakan Column Filter.
  3. Tangani data hilang (`missing value`) dengan Missing Value.
  4. Hapus baris duplikat dengan Duplicate Row Filter.
  5. Ubah tipe data kolom `Date` dari `String` menjadi `Date`.

4. Modeling dan Evaluasi

Pada tahap ini, dilakukan proses pemodelan untuk memprediksi status pengiriman berdasarkan data. Berikut langkah-langkahnya:

  1. Bagi data menjadi dua bagian (latih dan uji) dengan X-Partitioner.
  2. Latih model Random Forest dan Decision Tree menggunakan data latih.
  3. Lakukan prediksi pada data uji menggunakan kedua model.
  4. Evaluasi performa model dengan Scorer.

Hasil evaluasi menunjukkan bahwa model Random Forest memiliki akurasi 99.78%, sementara Decision Tree memiliki akurasi 99.64%. Kedua model memiliki performa yang sangat baik, namun model Random Forest sedikit lebih unggul.

5. Kesimpulan dan Rekomendasi

Kesimpulan

Dari hasil analisis, dapat disimpulkan bahwa Amazon perlu meningkatkan efisiensi pengiriman untuk pesanan yang dipenuhi sendiri oleh penjual. Pola penjualan menunjukkan bahwa pakaian dan sepatu adalah kategori produk yang paling sering dibeli.

Rekomendasi

Berikut beberapa rekomendasi untuk meningkatkan penjualan:

  • Perluas kerjasama dengan lebih banyak toko untuk produk pakaian dan sepatu.
  • Tawarkan promosi, diskon, dan bundling produk untuk kategori-kategori terlaris.
  • Tingkatkan layanan pengiriman untuk pesanan yang dipenuhi oleh penjual.
