Analisis Laporan Penjualan Amazon

Predictive modeling & fulfillment optimization.

1. Situation & Business Understanding

Ketidakteraturan data penjualan menyulitkan pemangku kepentingan dalam pengambilan keputusan strategis, merancang kampanye pemasaran, dan mendorong efisiensi pendapatan. Analisis ini dibangun untuk memetakan alur penjualan Amazon secara menyeluruh dan mengurai wawasan tersembunyi guna optimasi strategi di masa depan.

2. Task & Data Preparation

Dataset Sales Report.csv memuat 13 dimensi variabel. Tahap *engineering* dan persiapan data dieksekusi sepenuhnya melalui KNIME Analytics Platform:

  • Pembersihan kolom anomali (Unnamed: 22) via Column Filter.
  • Resolusi missing value dan filter baris duplikat.
  • Transformasi tipe data string menjadi struktur Date untuk analisis deret waktu.

3. Action & Modeling

Proses machine learning dikerahkan untuk memprediksi status pengiriman. Dataset dipartisi menjadi set latih dan uji menggunakan X-Partitioner. Dua algoritma dievaluasi secara komparatif:

  • Random Forest
  • Decision Tree

4. Result & Evaluation

Pengujian Scorer memvalidasi performa model yang luar biasa. Random Forest unggul dengan akurasi 99.78%, sementara Decision Tree mencatat 99.64%. Metrik ini memicu rekomendasi definitif:

  • Peningkatan efisiensi pengiriman untuk skema self-fulfillment oleh penjual.
  • Ekspansi kemitraan B2B berfokus pada kategori dengan volume tertinggi (pakaian & sepatu).
